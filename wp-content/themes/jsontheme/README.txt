JsonTheme es un tema moderno, optimizado para rendimiento, accesibilidad y SEO. Diseñado para ser ligero y compatible con Gutenberg, ideal para blogs, portafolios y más.

== Descripción ==
JsonTheme es un tema minimalista y versátil diseñado para ofrecer un rendimiento óptimo y una experiencia de usuario impecable. Este tema es completamente responsive, accesible y viene con soporte para:

- Gutenberg y bloques personalizados.
- Modos claro y oscuro.
- Opciones avanzadas en el personalizador de WordPress.
- Soporte para TailwindCSS y ACF.

Es perfecto para bloggers, desarrolladores y negocios que buscan un diseño limpio y moderno.

== Características ==
- **Diseño Responsivo:** Se adapta perfectamente a todos los dispositivos.
- **Optimización SEO:** Código limpio y marcado semántico.
- **Accesibilidad:** Cumple con los estándares WCAG 2.1.
- **Compatible con Gutenberg:** Personalización fácil y bloques modernos.
- **Opciones de Personalización:** Cambia colores, tipografía y más desde el Personalizador.
- **Compatibilidad con Plugins Populares:** Soporte para WooCommerce, ACF y WPML.

== Requisitos ==
- WordPress 6.0 o superior.
- PHP 8.2.20 o superior.
- Un navegador moderno (Chrome, Firefox, Edge, Brave, etc.).

== Instalación ==
1. Descarga el tema desde el repositorio oficial o desde tu panel de administración de WordPress.
2. Ve a **Apariencia > Temas > Añadir nuevo**.
3. Haz clic en "Subir tema" si tienes un archivo `.zip`, o busca "JsonTheme" en el repositorio.
4. Activa el tema.
5. Personaliza el tema desde **Apariencia > Personalizar**.
